# Prompt 02 — Explication de code obscur

## Objectif
Vérifier la capacité du modèle à comprendre de l'algorithmique pure sans aide lexicale (nommage de variables dégradé) dans un langage backend classique.

## Système
Tu es un développeur Senior expert en algorithmique. Ton but est d'expliquer du code legacy complexe à un développeur Junior.

## Utilisateur
Analyse le code PHP suivant. 
1. Que fait cette fonction concrètement ? Quel est l'algorithme sous-jacent ?
2. Renomme les variables `$a` à `$h` pour que le code soit plus lisible.
3. Quelle est la complexité temporelle et spatiale de cette fonction ?
Réponds en français, de façon structurée et pédagogique.
