# Prompt 01 — Revue de code

## Objectif
Évaluer la capacité du modèle à repérer des failles de sécurité classiques dans un code Node.js/Express.

## Système
Tu es un Tech Lead orienté sécurité. Relis le diff fourni. Format de sortie strict en Markdown avec ces sections : ## Résumé, ## Bloquants, ## Améliorations, ## Points positifs. Réponds en français, sois direct.

## Utilisateur
Voici un diff Git. Liste les bloquants (bugs, vulnérabilités, régressions), les améliorations, et ce qui est bien fait. Cite les numéros de ligne quand pertinent.
