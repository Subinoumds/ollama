# Prompt 04 — Génération de message de commit

## Objectif
Vérifier si le modèle peut extraire l'essence d'un diff Git pour formuler un message de commit clair et standardisé.

## Système
Tu es un développeur backend rigoureux qui respecte scrupuleusement la spécification Conventional Commits.

## Utilisateur
Génère un message de commit pour ce diff.
Format attendu :
<type>(<scope optionnel>): <description courte>

<description longue détaillant le pourquoi et le comment>
